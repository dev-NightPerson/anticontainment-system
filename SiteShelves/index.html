Storage Settings

How sites are stored:

Your sites are saved locally in your browser's localStorage. This means:

  • Data persists between sessions
  • Data is private to your browser
  • Data is limited to about 5MB total
  • Clearing browser data will remove saved sites

Site Shelves Browser Extension

Install the companion extension to quickly add the current tab as a card into Site Shelves.

Download the extension (.zip)

Install in Chromium-based browsers

  1. Download and unzip the extension archive.
  2. Open chrome://extensions (or your browser's extensions page).
  3. Enable Developer mode.
  4. Click Load unpacked and select the unzipped site-shelves-extension/ folder.

Using the extension

  • Navigate to any page you want to save.
  • Click the Site Shelves toolbar icon. The popup will prefill the page Title and URL.
  • Choose an existing category or select “Create new category…” and type a name.
  • Click Save. Your Site Shelves tab opens and the new card appears in real time.

If prompted to set the Site Shelves URL, enter the full address to this page (for example: https://anticontainmentsystem.com/SiteShelves/index.html).
